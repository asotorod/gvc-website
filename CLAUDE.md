# GVC Website - Claude Code Context

## Project Overview
Public marketing website for GVC School Bus - gvcbus.com

## Structure
- `/src` - Source files (React/Vite)
- `/public` - Static assets
- `index.html` - Entry point
- `vite.config.js` - Vite configuration

## Commands
```bash
npm run dev       # Local dev server
npm run build     # Production build
npm run preview   # Preview production build
```

## Tech Stack
- Framework: React + Vite
- Styling: CSS
- Deployment: Static hosting

## Style Guide
- Professional, trustworthy appearance (school bus company)
- Mobile-responsive design
- Fast loading (optimize images)
- Clear calls-to-action

## Rules
- Keep it simple and fast
- Ensure mobile responsiveness
- Optimize all images before adding
- Test across browsers before deployment
